# 사이42 저장소 작업 지침

- 이 저장소는 Vue 3 + FastAPI 모노레포다.
- 작업 전 관련 문서와 기존 코드를 먼저 확인한다.
- API 명세와 페이지 명세를 임의로 변경하지 않는다.
- 대규모 작업은 구현 전에 계획을 세운다.
- 서브 에이전트를 사용하지 않는다.
- 기능 구현 요청이 없는 경우 초기 설정 범위를 넘지 않는다.
- 실제 Secret을 코드나 문서에 기록하지 않는다.
- `.env` 파일을 커밋하지 않는다.
- Git commit과 push는 명시적 요청이 있을 때만 수행한다.
- 새로운 Production Dependency를 추가하기 전에 필요성을 확인한다.
- 기존 파일을 이유 없이 대규모 재포맷하지 않는다.
- 작업 후 실행한 명령과 검증 결과를 보고한다.
- 비밀번호는 과제 요구상 평문 저장 예정이지만 로그와 응답에는 노출하지 않는다.
- Frontend와 Backend 간 DTO와 API 계약은 명세를 따른다.
- 작업 범위를 넘어 실제 기능을 임의로 구현하지 않는다.
