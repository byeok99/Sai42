# 사이42 모바일 클릭형 와이어프레임

## 실행

`index.html`을 브라우저에서 열면 됩니다. VS Code의 Live Server를 사용해도 됩니다.

## 데모 계정

- 닉네임: `복숭아와호두`
- 비밀번호: `0420`

또는 첫 화면에서 **처음 등록**을 선택해 새 정보를 등록할 수 있습니다.

## 구현된 흐름

- 비밀번호 입력 또는 커플 등록
- 최초 입장 시 버튼형 데이트 성향 조사
- 날씨·축제 정보가 포함된 챗봇 화면
- 자연어 및 빠른 버튼으로 코스 변경
- 가상 지도 마커 시각화
- 코스 결정 및 현재 데이트 등록
- 장소별 좋아요와 다음 장소 진행
- 데이트 종료 후 한 줄 후기·평점 등록
- 랭킹보드 반영 및 다른 커플 코스 가져오기
- 과거 데이트 코스 조회

외부 API 대신 클릭 흐름을 확인할 수 있도록 가상 데이터를 사용한 UI 와이어프레임입니다.
